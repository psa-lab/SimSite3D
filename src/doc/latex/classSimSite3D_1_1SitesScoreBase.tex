\subsection{Detailed Description}
Base class for scoring two (aligned) sites. 

This class defines the interface required for a class to be able to be used to score two sites. A derived class can be used at either level of the scoring (template parameter 1 or 2 of \doxyref{Score\-Rigid\-Alignments}{p.}{classSimSite3D_1_1ScoreRigidAlignments})

notes on template parameters: align\_\-T: must be \doxyref{rigid\_\-align\_\-t}{p.}{classSimSite3D_1_1rigid__align__t} or derived class
